Перед запуском скрипта, выключите defender или прочие антивирусы

Для запуска скрипта требуется:
Python 3.12.x версии (За другие версии не могу ручаться)
Если вы используете python версии, которые меньше 3.10,
то устанавливайте зависимости через файл req_3.8.txt

Exe файлы находятся в директории EXE, а там уже выбираете версию windows, а затем разрядность, затем запускаете файл.

BAT файлы(для тех кто не хочет по каким либо причинам использовать exe файлы):
BAT файлы 
1.Cоздает виртуальное окружение
2.Обновляют pip
3.Устанавливает зависимости 
4.Запускает main.py

Для Windows 10-11:
Кликнуть по файлу "create venv_windows_10-11.bat", затем уже кликаете по файлу "run_main_windows_7_10-11.bat"
Для Windows 7:
Кликнуть по файлу "create venv_windows_7.bat", затем уже кликаете по файлу "run_main_windows_7_10-11.bat"


Приятного пользования :-)
